# Framework for the Interoperability of Cloud Platforms

Orleans is a cross-platform framework for building robust, scalable distributed applications. Supabase gives you a dedicated Postgres database to build your web, mobile and AI applications. IKVM is an implementation of Java for the Microsoft .NET platform. Espresso is an implementation of the Java Virtual Machine (JVM) that runs on the Truffle framework as part of GraalVM. Quarkus is a Cloud Native, Container First framework. Rascal meta-programming language.
